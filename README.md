## 知道某个重要信息，但repo中未列出？

欢迎贡献！请 PR！

需要注意的是本 repo 中使用了 pre-commit hook，请安装，并在有 bash 与 python3 的环境中
进行 commit。

在 pre-commit hook 的作用下，README.md 是自动生成的，故如果需要在 README.md 中修改，
请在 aux 文件夹下进行。

所有文件宽松执行一行八十字符（汉字字符宽度为两字符）的格式。
